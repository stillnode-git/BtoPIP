# BtoPIP
Auto render images (6 images per node for 360 panorama) from Blender to Pipmak Engine.

WORK IN PROGRESS
I wrote this script before and it needs to be rewritten entirely for blender 5.0.1

DOCUMENTATION
Create your world in blender then add empties (with names 1, 2, 3, 4 etc.) snapped directly to the floor corresponding to where you want your travel nodes.
The script will render all the images and default LUA scripts for you. Ready to use with Pipmak engine. (Drag and drop the project folder on the pipmak executable)

LINK TO PIPMAK
https://sourceforge.net/projects/pipmak/
